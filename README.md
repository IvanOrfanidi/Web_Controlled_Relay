## Web Controlled Relay
Проект управлением реле через Ethernet, Web интерфейс.

Конфигурация по COM порту AT командами.

### Перечень AT команд
* AT\r\n
* AT+REBOOT\r\n
* AT+GETIP\r\n
* AT+SETIP=192.168.1.1\r\n
* AT+GETMAC\r\n
* AT+SETMAC=00:12:34:56:78:90\r\n
* AT+GETPORT\r\n
* AT+SETPORT=8080\r\n
* AT+GETBUZ\r\n
* AT+SETBUZ=1\r\n
